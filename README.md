# Frontend Helper Pro

All-in-one VS Code extension for frontend developers.

Features:
- Auto rename paired HTML/XML/JSX tags
- Spell check comments, strings, Markdown, and HTML text
- Convert CSS units: px, rem, rpx, vw
- Quick Fix support
- Works with HTML, CSS, SCSS, React, Angular, Vue, Markdown

Perfect for frontend, AEM, React, Angular, and Vue developers.
